# 灯滅 / Tometsu

Tometsu is a short first-person horror mini-game built with Three.js.
Explore abandoned floors, purge ghosts with a fading flashlight, and survive the dark.

## Play Online

After GitHub Pages is enabled, the game can be played here:

`https://k3peta.github.io/tometsu/`

## Features

- First-person horror mini-game set in abandoned ruins
- Procedurally arranged floor layout
- Flashlight battery that gradually fades over time
- Ghost encounters with stylized spectral visuals
- Playable directly in the browser

## Controls

- Click: start
- Hold left mouse button: move
- Double click: flash
- Move mouse: look around
- `Esc`: release pointer lock
